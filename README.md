# ⚡ Installation

```bash
# Clone Repository
git clone https://github.com/your-username/zeroKnight-app.git

# Navigate to Project Folder
cd zeroKnight-app

# Install Dependencies
npm install

# Start Frontend
npm run dev
```

---

# ▶️ Run Backend Server

```bash
npm run server
```
